##########################
NODEJS: FILES AND FOLDERS
##########################

FS is a pre-determined NodeJs package with tools and methods, 
for example: writeFile is method

Reading, writing and appending files. See docs.

	const fs = require('fs');

	fs.readFile(); fs.writeFile(); fs.appendFile

Or the sync versions:
	
	fs.readFileSync; fs.writeFileSync() fs.appendFileSync()

See docs for detailed examples.

FOLDERS

fs.mkdir() or fs.mkdirSync()
fs.readdir() or fs.readdirSync
fs.rename() or fs.renameSync()
fs.rmdir() or fs.rmdirSync()

#######################
I.	CREATE A NEW FILE
#######################

To create a file in the filesystem, for example on the server or the local file system, through a Nodejs application:

The three arguments represent: a: file-path; b:data inside the file; c:callback function to display error or success.

	const fs = require('fs');

	fs.writeFile(a, b,c)

	fs.writeFile(__dirname + "/index.html", "<h1>Hello 				World</h1>", function(error){
    			if(error){return console.log("Error!");} 
				else{return console.log("Success!");}
	});

#####################
II.	DOWNLOAD A FILE 
#####################

For example, SAVE a picture from the internet in the local fs.

You require a protocol to deal with PROTOCOL you are going to use: https, http, file, ftp, etc. 

Use the .get method on that protocol variable.

	const fs = require("fs");
	const linkProtocol = require("https");

	linkProtocol.get(a, b)

a: URL or var that contains the URL
b: function to determine what to do with the downloaded data file

	const fileLocation = 						
		"https://wallpaperbrowse.com/media/images/3848765-			wallpaper-images-download.jpg";

	linkProtocol.get(fileLocation, writeToFs);

	function writeToFs(dataDownload){
		dataDownload.pipe(fs.createWriteStream(__dirname + 				"/new-picture.jpg"));
	};

The dataDownload is a random name for the data downloaded through the protocol. 

That data is read by from the download stream by as it becomes available AND writen to a destination writable stream with the fs.createWriteStream function.
